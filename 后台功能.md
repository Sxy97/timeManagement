### 1、新建公司
#### 请求URL:  
```
/back/addCompany
```

#### 请求方式: 
```
POST
```

#### 请求参数

|参数|是否必选|类型|说明|
|:-----|:-------:|:-----|:-----|
|name    |Y       |string  |公司名称|


#### 返回示例：

```javascript
{
    "code": 0,
    "data": "添加成功"
}
```

### 11、为人员分配公司及项目

#### 请求URL:
```
/back/PeopleAllocation
```

#### 请求方式: 
```
POST
```

#### 请求参数

|参数|是否必选|类型|说明|
|:-----|:-------:|:-----|:-----|
|peoid    |Y       |string|人员id|
|comid   |Y       |string|公司id|
|pros   |N      |数组|项目数组|

#### 请求示例：

```javascript
 const query = {
            peoid: 6,
            comid: 6,
            pros: [[1,6],[2,6],[3,6]]//[[项目id,公司id]]
        }
```
#### 返回示例：

```javascript
{
    "code": 0,
    "data": "分配成功"
}
```
